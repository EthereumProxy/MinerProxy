# MinerProxy
EthereumMinerProxy
下载后运行EthereumMinerProxy.exe
纯代理模式下不抽成，抽水代理模式下抽0.2%用于开发费用。
此版本暂时只支持SSL加密转发,为了安全,请使用SSL转发。
如需自定义证书请将cert.pem和key.pem放在运行软件的目录
将自动应用此目录下的证书,如不使用自定义证书,会使用默认证书.
